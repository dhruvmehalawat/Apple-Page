# Apple-Inspired Product Experience

A polished, high-end marketing website built with React and Vite, designed to feel like a premium Apple-style product showcase. This project combines cinematic storytelling, immersive animation, and interactive 3D visuals to create a modern front-end experience that feels both elegant and technically impressive.

## ✨ Highlights

This project showcases a full product storytelling journey with:

- A premium hero section with video content and strong visual hierarchy
- An interactive 3D MacBook Pro viewer with color and size switching
- Scroll-based animations and pinned sections powered by GSAP
- Performance-focused storytelling sections with layered motion graphics
- Responsive layout design optimized for desktop and mobile
- A sleek, minimal aesthetic inspired by modern Apple product marketing

## 🚀 Features

- Interactive product exploration with animated 3D models
- Dynamic color and size controls for the MacBook viewer
- Smooth scroll-triggered transitions and parallax-style motion
- Feature showcase cards with animated content reveal
- Responsive UI with mobile-friendly behavior
- Clean component-based architecture for scalability

## 🛠️ Skills & Technologies Used

### Frontend
- React 19
- Vite
- JavaScript (ES6+)
- CSS and Tailwind CSS

### 3D & Animation
- React Three Fiber
- React Three Drei
- Three.js
- GSAP
- ScrollTrigger

### State & UI Logic
- Zustand for global app state
- React hooks and reusable component structure
- Responsive design patterns with react-responsive
